# ThxProject
感謝の言葉やエピソードを投稿するサイト

## 作品URL
https://thxproject.herokuapp.com

## サイトの概要
1. テキストエリアに「感謝の言葉・エピソード」を入力・投稿する。(5文字以上)
2. 投稿されたエピソードに「いいね」をする。  
  ※他者、自身問わず。回数無制限。
3. Twitterと連携することでログインが可能。
4. ログインユーザーは以下の機能を閲覧可能。
   - 自身が投降したエピソードの閲覧
   - 自身がエピソードに「いいね」した回数
   - 自身のエピソードに「いいね」された回数
